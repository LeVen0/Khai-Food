## Опис проекту

Khai Food — повноцінний веб-застосунок для онлайн-замовлення їжі. Реалізовано каталог страв, кошик, особистий кабінет, систему бонусів і промокодів, оформлення замовлень з вибором способу доставки та оплати.

---

## Запуск проекту

### Спосіб 1 — Автоматичний
Двічі клацніть файл `start.bat` — він автоматично встановить залежності та запустить сайт.

### Спосіб 2 — Вручну через термінал

**Крок 1 — Бекенд:**
```bash
cd server
npm install
node index.js
```

**Крок 2 — Фронтенд (окремий термінал):**
```bash
cd client
npm install
npm run dev
```

Після запуску відкрийте браузер: **http://localhost:5173**

## Використані технології

**Фронтенд:**
- React 18, Vite
- React Router v6
- Axios
- CSS (власні змінні, адаптивна верстка)

**Бекенд:**
- Node.js + Express
- SQLite (sql.js)
- JWT авторизація
- bcryptjs

---

## Структура бази даних

7 таблиць: `users`, `categories`, `products`, `orders`, `order_items`, `promo_codes`, `bonus_transactions`

45+ позицій меню у 8 категоріях: бургери, піца, роли, снеки, напої, десерти, салати, комбо.

---

## Реалізований функціонал

- Каталог страв з фільтрацією та пошуком
- Кошик із збереженням у localStorage
- Реєстрація та авторизація (JWT)
- Особистий кабінет (профіль, замовлення, бонуси)
- Бонусна програма: рівні Бронза / Срібло / Золото
- Промокоди (відсоткова та фіксована знижка)
- Оформлення замовлення (доставка або самовивіз)
- Сторінка акцій
- Адаптивний дизайн для мобільних пристроїв
